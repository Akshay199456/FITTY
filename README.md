# FITTY

WELCOME TO THE FITTY APP 

A SOCIAL MEDIA APP FOR PEOPLE WHO ENJOY FASHION / STREETWEAR CULTURE. 

THIS APP WAS CREATED USING HTML, CSS, JAVASCRIPT, NODEJS &EXPRESS.
